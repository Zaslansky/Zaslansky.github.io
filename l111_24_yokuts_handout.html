Section A - Vowel Length

Vowel Shortening: [+syll] -> [-long] / __C{C,#}

… or, better, given that Yokuts syllables are CVC at their most complex…

Vowel Shortening: [+syll] -> [-long] / __[Coda C]

Sections B and C - Deletion or Epenthesis?

Vowel Deletion

If the UR for aorist passive is /-it/…

Vowel Deletion A: [+syll,-long] -> Ø / VC__CV

Vowel Deletion B: [+syll,-long] -> Ø / V__

or, less elegantly, [+syll,-long] -> Ø / V{C__CV , __}

UR /seːnit + al/ UR /seːnit + al/
Deletion se:ntal Shortening —
Shortening sental Deletion se:ntal
SR [sental] SR [seːntal]
Correct (Feeding) Incorrect (Counterfeeding)

… or you could choose to use …

Vowel Epenthesis

If the UR for aorist passive is /-t/…

Epenthesis: Ø -> [+syll, +high, +front] / C__C{C,#}

… or, better, again assuming a CVC syllable structure…

Epenthesis: Ø -> [+syll, +high, +front] / __[unsyllabified C]

UR /seːnt + ka/ UR /seːnt + ka/
Epenthesis seːnitka Shortening sentka
Shortening — Epenthesis senitka
SR [seːnitka] SR [seːnitka]
Correct (Bleeding) Incorrect (Counterbleeding)

Deletion works, but epenthesis is more elegant when involving syllables!
